## 블로그를 시작합니다

안녕하세요! 이 블로그는 **Astro**로 만들어졌습니다.

마크다운으로 편하게 글을 쓸 수 있고, 빌드 결과물은 정적 HTML이라 로딩이 빠릅니다.

### 기술 스택

- **Astro** — 정적 사이트 생성기
- **Tailwind CSS** — 유틸리티 기반 CSS
- **GitHub Pages** — 무료 호스팅

### 코드 예시

```javascript
const greeting = "Hello, World!";
console.log(greeting);
```
