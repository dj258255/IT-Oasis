# IT Oasis

A personal tech blog.

The desert of technology is vast and endless.
Between unfamiliar errors, tangled architectures, and sleepless debugging nights,
sometimes you just need a quiet place to sit and catch your breath.

That's what IT Oasis is -- a small resting spot I built for myself and anyone passing through.
A place where scattered knowledge settles into something clear.

**Live:** https://dj258255.github.io/IT-Oasis/

**Stack:** Astro 5 / Tailwind CSS 4 / GitHub Pages

---

개인 기술 블로그입니다.

기술의 사막은 끝없이 넓습니다.
낯선 에러, 꼬인 아키텍처, 밤새운 디버깅 사이에서
가끔은 조용히 앉아 숨을 고를 곳이 필요합니다.

흩어진 지식이 맑은 물처럼 고이는 곳.
